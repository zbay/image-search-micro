Image Searcher

Uses:

  1. Retrieve metadata for images hosted on imgur.com
  2. View recent image searches made using my domain

Sample Input:

https://url-shortener-micro.herokuapp.com/new/http://www.google.com

Sample Output:

{ "original_url": "http://www.bing.com", "shortened_url": "https://url-shortener-micro.herokuapp.com/tiny/c" }
